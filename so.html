Tipos de licencia de Windows

Existen diferentes tipos de licencia de Windows.

  • OEM → Ligadas a un dispositivo determinado, es decir que la licencia no podrá ser instalada ni utilizada en ningún otro dispositivo, bajo ninguna circunstancia. Sólo podrá ser utilizada en la computadora nueva que compramos.
  • Retail → La licencia Retail se diferencia de la licencia OEM fundamentalmente en el hecho de que podremos instalarla en cualquier dispositivo, siempre y cuando se respete la limitación de hacerlo en un solo equipo, es decir que no debemos instalar este software en más de un equipo simultáneamente.
  • Por volúmen → Ventas de licencias al por mayor.
  • Educativas → Licencias educativas que facilita Windows gracias a un convenio con el centro. Una vez se finalicen los estudios, las licencias se pierden.

¿Cómo funciona el sistema de arranque?

El sistema de arranque tanto en MBR como con particiones GPT funciona de una forma muy parecida, sólo existen pequeñas variaciones entre ellos.

FASE BIOS → Primera fase que se produce cuando encendemos el ordenador, son testeos y la búsqueda del sector de arranque.

  1. Corriente estable → power good
  2. FFFF0h → POST (power on self test) → test del hardware
  3. Pantalla donde sale la versión de la BIOS, etc.
  4. Bootstrap
  5. Lee desde donde arrancamos el SO
  6. Vamos a la tabla MBR → donde estarán las 4 particiones, la cual una será la activa, tiene el arranque. Además, tiene el AA55 para poder detectar en qué cilindro, sistema y sector se encuentra.
  7. Hay que tener en cuenta que en Linux, el sistema de arranque siempre estará en la primera partición libre: por ejemplo (/dev/sda1/). En cambio, en Windows puede ser que se ubique en segunda o tercer partición.

FASE BOOT → Ya sabemos donde está el gestor de arranque. El sistema operativo procede a arrancarse.

Particiones de forma correcta → 2 particiones 1 partición primaria para el sistema operativo y otra partición para los datos. Automáticamente nos creará un fichero de unos 100MB que contendrá donde se ubica el arranque:

  • Lilo → Arranque de versiones anteriores de Linux.
  • Grub2 → Arranque actual de Linux.
  • NTLDR → Arranque de Windows 95 a Windows XP.
  • Bootmgr → Arranque a partir de Windows 7 o posteriores.
  • Bitlocker → El programa para la encriptación de los datos.

UEFI → mismos pasos lo que en vez de 4 particiones, podremos tener hasta 128 particiones primarias. Además, crea un iMBR para tener una falsa compatibilidad con los discos antiguos y copia introduciendo el número en negativo en todos los ficheros.

FASE KERNEL → Carga de scripts y drivers. / Windows → Winload / Linux → Kernel Linux.
